Generating a New Application with OpenDaylight Startup Archetype

It seems the instructions on the OpenDaylight wiki is outdated as of 19 Aug 2018, and the startup maven archetype is now part of archetypes project. I think previously it was part of the OpenDaylight Controller. I couldn't find any instructions about how to generate maven archetype with the new archetype project, and sort of found my way around to create a new startup application. These are the steps I followed:

Get the archetypes project on your machine, and build the project. 
git clone https://github.com/opendaylight/archetypes.git
cd archetypes
mvn -nsu clean install
Move to a different directory which you want to create the project. Change the values in the following command for your own project. ("your.group.id", "YourAppName") 
mvn archetype:generate -DgroupId=your.group.id -DartifactId=YourAppName \
-DarchetypeGroupId=org.opendaylight.archetypes \
-DarchetypeArtifactId=opendaylight-startup-archetype
Then cd into newly created folder which will have the name: "YourAppName". And build the project. 
mvn -nsu clean install
After the build you can run karaf to see it is working. 
cd karaf/target/assembly/bin
./karaf
